Aturan main: 
1. Dalam game ini, user akan berperan sebagai penjual ice cream.
2. Di awal permainan, user akan menjual Ice Cream Cone seharga Rp.10k dengan cara meng-click objek truk.
3. User dapat membeli Auto-Clickers dan Buka Cabang Baru. 
   *Fitur Auto-Clickers akan membantu pemain mendapatkan uang tanpa harus mengklik objek truk ice cream. Uang akan bertambah sebanyak auto-clickers yang user    	beli, berlaku kelipatan.
   *Fitur Buka Cabang berguna untuk membuka cabang baru, sehingga pemasukan bertambah dari harga yang kita jual dikali total cabang, berlaku kelipatan.
4. User dapat meng-upgrade item yang ingin dijual, dengan syarat sebagai berikut: 
   a. Ice Cream cone Rp10k
      User sudah dapat menjual Ice Cream Cone diawal permainan. 
   b. Ice Cream Sundae Rp20k
      User harus memiliki uang minimal 100k untuk menjual Ice Cream Sundae.
   c. Banana Split Rp30k
      User harus memiliki uang minimal 200k untuk menjual Banana Split.
5. User tidak dapat membeli fitur atau upgrade item jika tidak memiliki cukup uang.
